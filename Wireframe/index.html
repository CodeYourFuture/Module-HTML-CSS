Why do developers need Git?

Aside version control, Git eases collaborative software development. It enables developers to simultaneously work on the same projects whereby the main file is submitted to the remote repository while each developer clones the repo copy to their local system, creates a branch, works on their local copy and then submit it to be merged to the main one on the remote.
